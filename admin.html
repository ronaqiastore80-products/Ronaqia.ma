Publication GitHub Pages

Le site public lit data/packs.json et data/config.json en statique. Après vos modifications locales, exportez les fichiers puis publiez-les sur GitHub.

  1. Modifiez vos packs et le numéro WhatsApp depuis cet admin
  2. Cliquez sur Exporter les données ci-dessous
  3. Remplacez data/packs.json et data/config.json dans votre dépôt
  4. Commit + push → GitHub Pages met à jour le site automatiquement

Fichiers générés : packs.json (tous les packs) et config.json (WhatsApp uniquement). Le fichier data/admin.json reste sur votre PC et n'est jamais publié.
